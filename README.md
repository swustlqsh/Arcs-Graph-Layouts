# ARCS-GRAPH-LAYOUTS
This demo is inspired by [TopicPanorama: a Full Picture of Relevant Topics](http://www.shixialiu.com/publications/TopicPanorama/paper.pdf).
I combine pie layout with force layout to visualize clusting data.
